Type of Lists

Ordered Lists

To make a list we use the tag's OL and LI, OL being the list and LI the content of that list

  1. To the left
  2. To the right
  3. Foward
  4. And beyond

The LI doenst need to be close with /li

If we want to change that type of list, from number to letter, we must use OL TYPE

  1. To the left
  2. To the right
  3. Foward
  4. And beyond

And to change the begginer number or letter we must use start=, ol type="" start="".

  1. To the left
  2. To the right
