External Links

To make a text turn into a link we must use tag A

Click here to view my linkedin profile.

To make that we dont exit the main site, we must add to the A tag this: target="_blank rel="external"

This is my GitHub profile

Internal Links

To make a internal link to my other website we simply create a new page and link to it, using the link methods

This is my second page

To make a better semantics, we can use REL, to say to the navigation system what page is next and what page is previous

We can use rel='nofollow' to websites that have no relations with our server, like a sponsorship site

Visit my sponsorship Worthouse

As instructed, we use _blank to go to other website, but there is a target that we can use to our on website, called _SELF, this is the right semantics.
